# Заметки

## Работа с mkdocs

Запуск командой:

```
mkdocs serve --livereload
```

После запуска доступен по адресу:

http://127.0.0.1:8000/ITMO_ICT_WebDevelopment_tools_2025-2026/

# Работа .venv

```
python3 -m venv .venv
source .venv/bin/activate
deactivate
```

## Работа с ресурсами

```
pip freeze > requirements.txt
pip install -r requirements.txt
```
